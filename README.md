# Postman
Postman
Задача:
Создать новую коллекцию в Postman и наполнить её методами для управления учетными записями пользователей (получить список, создать, обновить и удалить).

Этапы выполнения:

Ознакомьтесь на сайте https://reqres.in/ с описанием следующих методов: GET LIST USERS, POST CREATE, PUT UPDATE, DELETE
Создайте в postman новую коллекцию и пропишите в неё вызовы перечисленных выше методов
Пропишите для каждого вызова по 2 разных теста из доступных через группу snippets
Опционально: создайте переменную PATH для хранения в ней URL-адреса для вызова методов (https://reqres.in/) и замените ею адреса в перечисленных выше методах
